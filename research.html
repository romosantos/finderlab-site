EXP / 001
MEMORY · AGENTIC SYSTEMS

Quando um agente realmente precisa de memória?When does an agent really need memory?

Antes de adicionar uma camada de memória, queremos separar contexto útil, estado operacional e conhecimento que precisa sobreviver à conversa.Before adding a memory layer, we want to separate useful context, operational state and knowledge that needs to outlive the conversation.

HYPOTHESIS CONTEXT EVALUATION

Memória só agrega quando altera uma decisão futura de forma verificável.Memory only adds value when it changes a future decision in a verifiable way.

LAB NOTE00107.2026

Um agente em produção não falha só porque o modelo erra.An agent in production doesn't fail only because the model gets it wrong.

Ele falha quando ninguém definiu qual sinal importa, quando o contexto chega quebrado, quando a ação não tem limite ou quando não existe caminho para revisão humana. Modelo é uma parte da equação. Sistema é a outra.It fails when no one defined which signal matters, when context arrives broken, when action has no boundary or when there is no path for human review. The model is one part of the equation. The system is the other.
