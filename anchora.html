A viradaThe shift

Output vira relatório. Outcome muda a carreira.Output becomes a report. Outcome changes a career.

A pergunta que separa uma ferramenta de um dashboard é o teste 'e daí?'. Marcar '15 1:1s feitos' é output. Saber que a pessoa destravou é outcome. A Anchora foi construída para o segundo.The question that separates a tool from a dashboard is the 'so what?' test. Logging '15 1:1s done' is output. Knowing the person unlocked is outcome. Anchora is built for the second.
